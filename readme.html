﻿Электронный учебник «Информатика» содержит элементы математической разметки MathML и векторной графики SVG, что накладывает требование использовать для просмотра документов браузер Mozilla FireFox версии 3.6 и выше (или Internet Explorer версии 6.0 и выше с плагином Math Player, который можно найти по адресу http://www.dessci.com/en/products/mathplayer/).

Для исполнения скриптов JavaScript, внедренных в документы электронного учебника, рекомендуется настроить в браузере исполнение активных сценариев (разрешить исполнение активных сценариев). Соответствующие инструкции по настройке браузеров можно прочитать на странице http://www.plantarium.ru/page/help/topic/javascript.html.
