Электронный учебник «Информатика» содержит элементы математической разметки MathML и векторной графики SVG.

Для исполнения скриптов JavaScript, внедренных в документы электронного учебника, рекомендуется настроить в браузере исполнение активных сценариев (разрешить исполнение активных сценариев). Соответствующие инструкции по настройке браузеров можно прочитать на странице http://www.plantarium.ru/page/help/topic/javascript.html.
