8.10.8. Сравнение методов сортировки

Пусть n — количество элементов в сортируемом массиве, С и M — количество необходимых для сортировки сравнений ключей и операций присваивания соответственно. Для всех трех простых методов сортировки в таблице 8.10.8.1 приведены аналитические формулы оценки их эффективности:

Таблица 8.10.8.1. Эффективность простых методов сортировки
Метод Минимальная оценка Средняя оценка Максимальная оценка
Сортировка простыми включениями C = n - 1 , M = 2 ⁢ ( n - 1 ) C = 1 4 ⁢ ( n 2 + n - 2 ) , M = 1 4 ⁢ ( n 2 - 9 ⁢ n - 10 ) C = 1 2 ⁢ ( n 2 - n - 2 ) , M = 1 2 ⁢ ( n 2 + 3 ⁢ n - 4 )
Сортировка простым выбором C = 1 2 ⁢ ( n 2 - n ) , M = 3 ⁢ ( n - 1 ) C = 1 2 ⁢ ( n 2 - n ) , M = n ⁢ ( ln ⁡ n + 0.57 ) C = 1 2 ⁢ ( n 2 - n ) , M = n 2 4 + 3 ⁢ ( n - 1 )
Сортировка пузырьком C = 1 2 ⁢ ( n 2 - n ) , M = 0 C = 1 2 ⁢ ( n 2 - n ) , M = 3 4 ⁢ ( n 2 - n ) C = 1 2 ⁢ ( n 2 - n ) , M = 3 2 ⁢ ( n 2 - n )

Для усовершенствованных методов сортировки существуют лишь асимптотические оценки вычислительной сложности: для сортировки Шелла (включениями с убывающим приращением) имеет место оценка c i ⁢ n 1.8 , а в случаях же пирамидальной и быстрой сортировки — c i ⁢ n ⁢ log ⁡ n .

Приведенные асимптотические оценки позволяют классифицировать методы сортировки на квадратичные (субквадратичные) и логарифмические ( n ⁢ log ⁡ n ) .

Данные таблицы 8.10.7.2 позволяют утверждать, что при сортировке одномерного массива целых чисел в Компонентном Паскале:

  1. из простых методов сортировки случайных массивов лучшую эффективность демонстрирует метод сортировки простыми включениями, а худшую — метод пузырька;
  2. из простых методов сортировки массивов с обратным порядком лучшую эффективность демонстрирует метод сортировки простым выбором, а худшую — шейкер-сортировка;
  3. при сортировке упорядоченной последовательности худшие показатели при использовании метода пузырька и сортировки выбором;
  4. быстрая сортировка превосходит пирамидальную сортировку в отношении 1 к 3. Она сортирует массив с элементами, расположенными в обратном порядке, практически так же, как уже рассортированный.

Для практических целей было бы полезно получить оценки значений коэффициентов c i опытным путем. Следует учитывать и вспомогательные операции, используемые при сортировке массивов, такие, как управление циклами, вызов процедур и т.д., а также использование вспомогательных переменных. Очевидно, что на эффективность алгоритма сортировки будет оказывать влияние и тип элементов массива. Например, сортировка последовательности записей потребует больших вычислительных ресурсов, нежели сортировка числовой последовательности. Наконец, не последнюю роль играет язык программирования, на котором реализован алгоритм сортировки.

    • монотонно возрастающих чисел;

    • с равными элементами.
